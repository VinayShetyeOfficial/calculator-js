## 🚀 Getting Started

### Prerequisites

- Modern web browser
- Audio-enabled device

### Installation

1. Clone the repository:
   ```sh
   git clone https://github.com/VinayShetyeOfficial/Calculator-JS.git
   ```
2. Navigate to the project directory:
   ```sh
   cd JS-Calculator
   ```
3. Open in browser:
   ```sh
   open index.html
   ```
